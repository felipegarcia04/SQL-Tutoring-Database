# SQL-Tutoring-Database
Developed a relational database system for a tutoring non-profit using SQL. The database is able to store information such as student names, parent/guardian names, and progress report scores.  This project demonstrates database design, query building, storing procedures, and views .
## Business Problem
Tutoring organizations need an effective way to manage student records, parent/guardian information, subjects, and performance reports. Without developing a structured database, information can become fragmented and innacurate.
## Skills Demonstrated
- Database Management
- Joins
- Views
- Stored Procedures
- Queries
- Primary and Foreign keys
- Relational Data Modeling
